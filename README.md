# 陳賢宗老師｜GitHub Pages 上線版

這份壓縮包可直接上傳到 GitHub 作為靜態網站使用。

## 檔案內容
- `index.html`：首頁主檔
- `README.md`：部署說明

## GitHub Pages 上線步驟
1. 到 GitHub 建立一個新的 repository。
2. 將壓縮包解開後，把 `index.html` 上傳到 repository 根目錄。
3. 進入 GitHub 專案頁面。
4. 點選 `Settings`。
5. 點選左側 `Pages`。
6. 在 `Build and deployment` 中：
   - `Source` 選 `Deploy from a branch`
   - `Branch` 選 `main`
   - 資料夾選 `/ (root)`
7. 按下 `Save`。
8. 等待 1 到 3 分鐘後，GitHub 會產生網站網址。
